# Word Clash Frontend

React-based frontend for the Word Clash multiplayer word game.

## Features

- Real-time multiplayer gameplay with Socket.io
- Beautiful gradient UI with TailwindCSS
- Sound effects with mute toggle
- Confetti animations for victories
- Responsive design

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

App will run on http://localhost:3000

## Building for Production

```bash
npm run build
```
