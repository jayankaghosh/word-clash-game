# Assets Folder

Place your app assets here:

## Required Files

- **icon.png** (1024x1024 px) - App icon
- **splash.png** (1242x2436 px) - Splash screen
- **adaptive-icon.png** (1024x1024 px) - Android adaptive icon
- **favicon.png** (48x48 px) - Web favicon

## Notes

For development, these files are optional. Expo will generate default icons if they're missing.

For production, you should create proper assets matching your app's branding.
