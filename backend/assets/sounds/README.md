# Sound Effects

The following WAV files are in this directory:

## Sound Effects:
1. **click.wav** - Quick click/tap sound for buttons (also used for submit)
2. **success.wav** - Pleasant "ding" or achievement sound  
3. **join.wav** - Player joining sound (welcoming chime)
4. **start.wav** - Game starting fanfare (also used for roundStart)
5. **tick.wav** - Timer countdown tick (when < 5 seconds)
6. **reveal.wav** - Letters revealing sound (magical/whoosh)
7. **win.wav** - Round victory sound (short celebration)
8. **lose.wav** - Round loss sound (sad trombone/descending notes)
9. **gameWin.wav** - Final game victory fanfare (epic celebration)
10. **gameLose.wav** - Final game loss sound (game over)
11. **error.wav** - Error/invalid action sound (buzz/negative)

## Background Music:
12. **bg.mp3** - Background music that loops continuously throughout the game

## Sound Mapping:
- `click` → click.wav
- `submit` → click.wav (reused)
- `start` → start.wav
- `roundStart` → start.wav (reused)
- Background music → bg.mp3 (loops automatically at 20% volume)

## Recommended Sound Types:
- **Duration:** 0.1 - 2 seconds (keep them short)
- **Format:** MP3
- **Volume:** Normalized to -14 LUFS or similar
- **Style:** 8-bit retro, modern UI sounds, or game-style SFX
